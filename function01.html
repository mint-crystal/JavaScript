함수(Function)

선언적 함수

function 키워드에 함수명을 작성하여 사용하는 방법
함수를 메모리에 바로 올려놓기 때문에 코드의 순서에 상관없이 바로 호출 가능

function 함수명([매개변수]){
처리로직;
[return 값;]
}

익명 함수

function에 함수명을 작성하지 않고 변수에 함수를 넣어 작성하는 방법
함수 선언 이후에만 사용 가능(script 내부에서만)

var 변수명 = function([매개변수]){ 처리로직; [return 값;] }

스스로 동작하는 함수(호출없이 바로 실행되는 구조)

function에 함수명을 작성하지 않고 작성

(function(){
처리 로직;
//리턴값 없음(스스로 호출하고 끝나기 때문)
})();
